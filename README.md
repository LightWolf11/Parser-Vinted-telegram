# Vinted Bot

Парсер вещей связанный с Vinted, включающее Telegram-бота и систему уведомлений.

## Структура проекта

### Основные компоненты

- **bot.py** - главный Telegram бот для управления Vinted профилем
- **vinted_selenium.py** - интеграция Selenium для взаимодействия с веб-версией Vinted
- **check_proxy.py** - утилита для проверки работоспособности прокси
- **config.toml** - конфигурационный файл с параметрами приложения

### Папки

- **proxies/** - модуль для работы с прокси-серверами
  - `proxy_factory.py` - фабрика для создания прокси подключений
  - `proxy.py` - класс для управления прокси

- **vinted-api-wrapper/** - обертка для API Vinted
  - Модули для работы с эндпоинтами и моделями данных

- **Vinted-Notifications/** - система уведомлений о товарах
  - **core.py** - основная логика
  - **db.py** - работа с БД
  - **pyVintedVN/** - утилиты для работы с API
  - **rss_feed_plugin/** - плагин для генерации RSS-ленты
  - **web_ui_plugin/** - веб-интерфейс для управления
  - **telegram_bot_plugin/** - интеграция с Telegram

## Установка

1. Клонировать репозиторий
2. Установить зависимости:
```bash
pip install -r requirements-bot.txt
```
3. Настроить config.toml с параметрами:
   - `tel` - токен Telegram бота
   - `proxy_string` - URL прокси (опционально)
   - `stripe_secret_key` - ключ Stripe (опционально)

4. Запустить бот:
```bash
python bot.py
```

## Зависимости

- python-telegram-bot - Telegram Bot API
- vinted-api-wrapper - API для Vinted
- selenium - автоматизация браузера
- requests - HTTP запросы
- PySocks - поддержка SOCKS прокси

## Особенности

- Поддержка прокси-серверов
- Интеграция со Stripe
- Система уведомлений о новых товарах
- Веб-интерфейс для управления настройками
- Телеграм бот для взаимодействия
- RSS-лента товаров
